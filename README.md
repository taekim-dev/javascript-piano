# JavaScript Piano

Compose and play piano music with a vanilla JavaScript and Vue 3 powered web application.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
